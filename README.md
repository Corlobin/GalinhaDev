# GalinhaDev
__'Author'__ = 'Antonio'

O jogo consiste no famoso desenho animado do papaleguas onde o papaleguas corre do coyote para salvar sua vida, que quase sempre, nunca é pego.
O personagem consiste no coyote que deve coletar moedas no decorrer do cenario até conseguir chegar no final e capturar o coyote. Quem coleta
mais moedas é o mais recordista e existem moedas de diferentes cores.
